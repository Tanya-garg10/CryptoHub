# Security Policy

## Supported Versions

We currently support the latest version of the project. Please ensure you are using the most recent release or the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please report it to us immediately.

### How to Report
Since this is an open-source project, you can report vulnerabilities by:

1. **Opening an Issue:** Please describe the issue clearly. If it is sensitive, please mark it accordingly.
2. **Email:** Contact the repository maintainer directly (if an email is listed on their profile).

We will try to review and address the issue as soon as possible.
